## 权限说明

本库要求SDK最低版本为16，代码中配置如下
```
android {
        ...
        defaultConfig {
            ...
            minSdkVersion 16
            ...
        }
        ...
    }
```


## build.gradle设置
```
dependencies {
    ...
    compile 'com.zevol.library:ZevolLibrary:1.0.3'
    ...
}
```
